# 퀀트 투자 대시보드 MVP PRD
**v0.1 | 2026.03.20**

---

## 1. 개요

**제품명:** QuantRadar (가칭)

**목표:** 감이 아닌 규칙으로 한국/미국 주식을 모니터링하고, 매매 신호를 실시간으로 받는 개인용 퀀트 대시보드

**배경:**
개인 투자자는 수백 개 종목을 매일 직접 확인하는 게 불가능하다. 메달리온 펀드의 핵심 철학인 "규칙 기반 의사결정"을 개인 투자자 수준에서 구현한다. 브라우저에서 바로 쓰고, 코딩 없이 전략을 설정한다.

**타겟 사용자:** 데이터 기반으로 투자하고 싶지만 코딩은 못 하는 개인 투자자 (1인용)

**성공 지표:**
- 하루 1번 이상 대시보드 접속
- 신호 발생 후 실제 매매 연결 여부 추적 가능
- 백테스트 수익률이 같은 기간 코스피/S&P 500 대비 우상향

## 2. 사용자 스토리

2. Ethan은 새 전략을 설정할 때 코딩 없이 조건을 선택하고 과거 성과를 즉시 본다
3. Ethan은 보유 중인 종목이 매도 신호에 걸리면 알림을 받는다

## 3. 기능 명세

### P0 (MVP 필수)

| 기능 | 설명 |
|------|------|
| 종목 유니버스 설정 | 코스피200 + S&P500 중 관심 종목 선택 |
| 전략 빌더 | 조건 선택 방식 (PER, RSI, 이동평균, 거래량 등) |
| 백테스트 | 설정한 전략을 과거 3년 데이터에 적용, 수익률 그래프 출력 |
| 실시간 신호 대시보드 | 오늘 매수/매도 조건에 걸린 종목 리스트 |
| 신호 알림 | 조건 달성 시 이메일 또는 카카오톡 알림 |

### P1 (이후 추가)

| 기능 | 설명 |
|------|------|
| 포트폴리오 추적 | 실제 보유 종목 수익률 추적 |
| 전략 성과 비교 | 내 전략 vs 코스피/S&P 500 벤치마크 |
| 멀티 전략 | 전략 2개 이상 동시 운영 |

### Out of Scope (이번엔 제외)

- 자동 매매 (증권사 API 연결)
- AI 종목 추천
- 소셜 기능 (다른 사람 전략 공유)
- 선물/옵션/암호화폐

## 4. 사용자 플로우

```
대시보드 진입
→ 오늘의 신호 종목 확인 (메인 화면)
→ 종목 클릭 → 상세 차트 + 신호 근거 확인
→ 매매 결정 (수동)

별도 플로우:
전략 빌더 진입
→ 조건 선택 (드롭다운)
→ 백테스트 실행
→ 결과 확인 후 전략 저장 또는 수정
```

---

## 5. 기술 요구사항

**데이터 소스:**
- 한국 주식: KIS Developers API (한국투자증권, 무료)
- 미국 주식: Yahoo Finance API 또는 Polygon.io
- 재무 데이터(PER 등): 네이버 금융 크롤링 또는 FnGuide

**프론트엔드:** 웹 브라우저 기반, 모바일에서도 볼 수 있게

**백엔드:** Python (데이터 처리, 신호 계산)

**알림:** 카카오 알림톡 또는 이메일

**성능:**
- 신호 업데이트: 장 중 30분 간격
- 백테스트 결과 출력: 10초 이내

---

## 6. 핵심 화면

1. **메인 대시보드:** 오늘의 매수/매도 신호 종목, 전략별 현황 요약
2. **전략 빌더:** 조건 선택 UI + 백테스트 결과 그래프
3. **종목 상세:** 차트, 신호 발생 이력, 재무 지표
4. **알림 설정:** 알림 조건, 수신 방법 설정

## 7. 일정

| 마일스톤 | 기간 | 산출물 |
|----------|------|--------|
| 데이터 연결 + 백테스트 엔진 | 1-2주 | 종목 데이터 수집, 전략 계산 로직 |

| 알림 연동 | 4-5주 | 이메일/카카오 알림 |
| 테스트 + 개인 운용 시작 | 5-6주 | 실사용 버전 |
